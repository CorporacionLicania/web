CURSO DE INTRODUCCIÓN A LA CIENCIA DE DATOS

En un mundo cada vez más digitalizado, vamos produciendo datos como huellas con nuestras acciones. Al visitar una página web, al publicar en nuestras redes sociales, o al consultar los servicios de salud estamos produciendo datos que son recopilados por entes privados y gubernamentales, y que informan las decisiones que estos toman, desde la adopción de medidas sanitarias para contener una enfermedad, hasta ofrecernos un producto a través de publicidad.

Para bien o para mal, parece cada vez más difícil escapar a este mundo digitalizado. Ante este panorama, cabe preguntarnos ¿cómo sacar provecho de los datos que producimos para comprender las realidades sociales en que vivimos y poder transformarlas? Este curso está orientado a ofrecer herramientas conceptuales y habilidades digitales que permitan usar datos para ayudar a la comprensión de las complejas realidades sociales en que vivimos.

¿Qué aprenderemos en este curso?

  • Fundamentos de programación en Python
  • Manejo básico de bases de datos
  • Diseño y aplicación de encuestas
  • Descripciones y análisis estadístico de datos
  • Visualización de datos y comunicación de resultados

¿Qué necesito para participar de este curso?

  • Tener acceso a un computador con sistema operativo Linux, Windows o MacOS
  • Tener acceso a una conexión de internet estable
  • Tener disponibilidad de 2 a 3 horas semanales para las clases virtuales
  • Tener disponibilidad de 1 o 2 horas semanales de trabajo independiente
  • Muchas ganas de aprender

¿Cómo puedo postularme para participar

Debes registrarte en el siguiente formulario. El curso es gratuito pero la aceptación de la postulación está sujeta a disponibilidad de cupos. Para la asignación de los cupos se tendrá prioridad con las personas que pertenezcan a organizaciones sociales del municipio de Marinilla.
